Tabs

When in the textarea, press tab to insert a tab

Tab Indent line(s)

When in the textarea, select some text, and press tab to indent line(s) the selected text belongs to

Tab UN-Indent line(s)

When in the textarea, select some text, and press shift-tab to UN-indent line(s) the selected text belongs to

This will remove tabs from the begining of the line(s)

Auto Indent on return

If the line you are currently on in the textarea is indented, hitting return will automatically add the same indentation to the next line
